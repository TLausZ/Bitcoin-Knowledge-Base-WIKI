# BIP-0382: Segwit Output Script Descriptors

**Status:** established
**Last updated:** 2026-06-08
**Sources:** [[bip-0382]]

Layer: Applications · Typ: Informational · Status: Deployed · Datum: 2021-06-27
**Autoren:** Pieter Wuille <pieter@wuille.net>, Ava Chow <me@achow101.com>

## Summary

This document specifies wpkh(), and wsh() output script descriptors. wpkh() descriptors take a key and produces a P2WPKH output script. wsh() descriptors take a script and produces a P2WSH output script.
