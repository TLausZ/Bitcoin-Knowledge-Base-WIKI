# BIP-0177: Redefine Bitcoin's Base Unit

**Status:** emerging
**Last updated:** 2026-06-07
**Sources:** [[bip-0177]]

Typ: Informational · Status: Draft · Datum: 2025-04-23
**Autoren:** John Carvalho

## Summary

This BIP proposes redefining the commonly recognized "bitcoin" unit so that the base unit becomes the primary reference unit. Under this proposal, one bitcoin is defined as that indivisible base unit, eliminating the convention of synthetic decimal places. The ledger counts discrete integral units; the decimal point is a human-imposed abstraction.
