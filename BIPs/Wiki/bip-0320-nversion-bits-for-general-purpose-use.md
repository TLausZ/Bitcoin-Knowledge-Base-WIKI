# BIP-0320: nVersion bits for general purpose use

**Status:** emerging
**Last updated:** 2026-06-08
**Sources:** [[bip-0320]]

Typ: Specification · Status: Draft · Datum: 2018-03-01
**Autoren:** BtcDrak <btcdrak@gmail.com>

## Summary

This BIP reserves 16 bits of the block header nVersion field for general purpose use and removes their meaning for the purpose of version bits soft-fork signalling. The key words "MUST", "MUST NOT", "REQUIRED", "SHALL", "SHALL NOT", "SHOULD", "SHOULD NOT", "RECOMMENDED", "MAY", and "OPTIONAL" in this document are to be interpreted as described in RFC 2119.
