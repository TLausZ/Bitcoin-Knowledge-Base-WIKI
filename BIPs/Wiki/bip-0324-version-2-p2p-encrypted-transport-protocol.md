# BIP-0324: Version 2 P2P Encrypted Transport Protocol

**Status:** established
**Last updated:** 2026-06-07
**Sources:** [[bip-0324]]

Layer: Peer Services · Typ: Specification · Status: Deployed · Datum: 2019-03-08
**Autoren:** Dhruv Mehta, Tim Ruffing, Jonas Schnelli, Pieter Wuille

## Summary

This document proposes a new Bitcoin P2P transport protocol, which features opportunistic encryption, a mild bandwidth reduction, and the ability to negotiate upgrades before exchanging application messages.
