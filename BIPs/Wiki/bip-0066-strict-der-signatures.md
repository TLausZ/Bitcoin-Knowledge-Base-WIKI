# BIP-0066: Strict DER signatures

**Status:** established
**Last updated:** 2026-06-07
**Sources:** [[bip-0066]]

Layer: Consensus (soft fork) · Typ: Specification · Status: Deployed · Datum: 2015-01-10
**Autoren:** Pieter Wuille

## Summary

This document specifies proposed changes to the Bitcoin transaction validity rules to restrict signatures to strict DER encoding.
