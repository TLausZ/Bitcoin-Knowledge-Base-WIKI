# BIP-0133: feefilter message

**Status:** established
**Last updated:** 2026-06-07
**Sources:** [[bip-0133]]

Layer: Peer Services · Typ: Specification · Status: Deployed · Datum: 2016-02-13
**Autoren:** Alex Morcos

## Summary

Add a new message, "feefilter", which serves to instruct peers not to send "inv"'s to the node for transactions with fees below the specified fee rate.
