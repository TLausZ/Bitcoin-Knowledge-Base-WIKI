# BIP-0122: URI scheme for Blockchain references / exploration

**Status:** emerging
**Last updated:** 2026-06-07
**Sources:** [[bip-0122]]

Layer: Applications · Typ: Specification · Status: Draft · Datum: 2015-08-29
**Autoren:** Marco Pontello

## Summary

This BIP proposes a URI scheme for looking up blocks, transactions, and addresses on a Blockchain explorer, or for making proper Blockchain references in general.
