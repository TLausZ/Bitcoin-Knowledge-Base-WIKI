# BIP-0433: Pay to Anchor (P2A)

**Status:** emerging
**Last updated:** 2026-06-08
**Sources:** [[bip-0433]]

Layer: Applications · Typ: Informational · Status: Draft · Datum: 2025-12-08
**Autoren:** Gregory Sanders <gsanders87@gmail.com>

## Summary

This document describes a new standard output script called Pay to Anchor (P2A) and prescribes making spending of this output type standard. This output is "keyless" meaning no signature or witness data at all are required to spend this output typea.
