# BIP-0101: Increase maximum block size

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0101]]

Layer: Consensus (hard fork) · Typ: Specification · Status: Closed · Datum: 2015-06-22
**Autoren:** Gavin Andresen

## Summary

This BIP proposes replacing the fixed one megabyte maximum block size with a maximum size that grows over time at a predictable rate.
