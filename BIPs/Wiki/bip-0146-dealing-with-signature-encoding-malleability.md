# BIP-0146: Dealing with signature encoding malleability

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0146]]

Layer: Consensus (soft fork) · Typ: Specification · Status: Closed · Datum: 2016-08-16
**Autoren:** Johnson Lau, Pieter Wuille

## Summary

This document specifies proposed changes to the Bitcoin transaction validity rules to fix signature malleability related to ECDSA signature encoding.
