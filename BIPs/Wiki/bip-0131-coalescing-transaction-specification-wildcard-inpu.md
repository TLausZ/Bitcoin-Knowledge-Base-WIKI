# BIP-0131: \"Coalescing Transaction\" Specification (wildcard inputs)

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0131]]

Layer: Consensus (hard fork) · Typ: Specification · Status: Closed · Datum: 2015-11-30
**Autoren:** Chris Priest

## Summary

This specification defines a new type of transaction that supplements (not replaces) normal "non coalescing" bitcoin transactions.
