# BIP-0156: Dandelion - Privacy Enhancing Routing

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0156]]

Layer: Peer Services · Typ: Specification · Status: Closed · Datum: 2017-06-09
**Autoren:** Brad Denby, Andrew Miller, Giulia Fanti, Surya Bakshi, Shaileshh Bojja Venkatakrishnan, Pramod Vi...

## Summary

Bitcoin's transaction spreading protocol is vulnerable to deanonymization attacks. Dandelion is a transaction routing mechanism that provides formal anonymity guarantees against these attacks. Transactions travel along a randomly selected path during the "stem phase" before being diffused during the "fluff phase."
