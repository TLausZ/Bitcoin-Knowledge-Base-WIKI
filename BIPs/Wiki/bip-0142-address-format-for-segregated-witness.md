# BIP-0142: Address Format for Segregated Witness

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0142]]

Layer: Applications · Typ: Specification · Status: Closed · Datum: 2015-12-24

## Summary

This BIP describes new types of Bitcoin address to support native segregated witness transactions with 20-byte and 32-byte program.
