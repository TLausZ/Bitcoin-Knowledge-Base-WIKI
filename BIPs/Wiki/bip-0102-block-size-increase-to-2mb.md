# BIP-0102: Block size increase to 2MB

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0102]]

Layer: Consensus (hard fork) · Typ: Specification · Status: Closed · Datum: 2015-06-23
**Autoren:** Jeff Garzik

## Summary

Simple, one-time increase in total amount of transaction data permitted in a block from 1MB to 2MB.
