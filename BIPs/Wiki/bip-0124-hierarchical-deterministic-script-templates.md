# BIP-0124: Hierarchical Deterministic Script Templates

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0124]]

Layer: Applications · Typ: Informational · Status: Closed · Datum: 2015-11-20
**Autoren:** Eric Lombrozo, William Swanson

## Summary

This BIP defines a script template format that can be used by wallets to deterministically generate scripts with specific authorization policies using the key derivation mechanism defined in BIP32.
