# BIP-0385: raw() and addr() Output Script Descriptors

**Status:** established
**Last updated:** 2026-06-08
**Sources:** [[bip-0385]]

Layer: Applications · Typ: Informational · Status: Deployed · Datum: 2021-06-27
**Autoren:** Pieter Wuille <pieter@wuille.net>, Ava Chow <me@achow101.com>

## Summary

This document specifies raw() and addr() output script descriptors. raw() encapsulates a raw script as a descriptor. addr() encapsulates an address as a descriptor.
