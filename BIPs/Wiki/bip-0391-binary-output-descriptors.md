# BIP-0391: Binary Output Descriptors

**Status:** speculative
**Last updated:** 2026-06-08
**Sources:** [[bip-0391]]

Layer: Applications · Typ: Specification · Status: Closed · Datum: 2026-01-27
**Autoren:** SeedHammer <team@seedhammer.com>

## Summary

This document proposes an efficient container format for encoding an output descriptor as defined by BIP 380 and its extensions. Based on key-value maps as defined by BIP 174, the encoding supports descriptor and key metadata.
