# BIP-0384: combo() Output Script Descriptors

**Status:** established
**Last updated:** 2026-06-08
**Sources:** [[bip-0384]]

Layer: Applications · Typ: Informational · Status: Deployed · Datum: 2021-06-27
**Autoren:** Pieter Wuille <pieter@wuille.net>, Ava Chow <me@achow101.com>

## Summary

This document specifies combo() output script descriptors. These take a key and produce P2PK, P2PKH, P2WPKH, and P2SH-P2WPKH output scripts if applicable to the key.
