# BIP-0126: Best Practices for Heterogeneous Input Script Transactions

**Status:** emerging
**Last updated:** 2026-06-07
**Sources:** [[bip-0126]]

Typ: Informational · Status: Draft · Datum: 2016-02-10
**Autoren:** Kristov Atlas

## Summary

When a Bitcoin transaction contains inputs referencing previous outputs sent to different addresses, personally identifiable information leaks into the blockchain. This document proposes best practice guidelines to minimize uncontrolled disclosure of PII by defining standard forms for transactions containing heterogeneous input scripts.
