# BIP-0172: Define Bitcoin Subunits as Satoshis

**Status:** emerging
**Last updated:** 2026-06-07
**Sources:** [[bip-0172]]

Layer: Applications · Typ: Informational · Status: Draft · Datum: 2025-05-01
**Autoren:** OceanSlim

## Summary

This BIP proposes to formally define and standardize Bitcoin's smallest indivisible unit (1/100,000,000 of a bitcoin) as "satoshi" (singular) or "satoshis" (plural), with "sats" as the standard abbreviated form.
