# BIP-0083: Dynamic Hierarchical Deterministic Key Trees

**Status:** speculative
**Last updated:** 2026-06-07
**Sources:** [[bip-0083]]

Layer: Applications · Typ: Specification · Status: Closed · Datum: 2015-11-16
**Autoren:** Eric Lombrozo

## Summary

This BIP defines a scheme for key derivation that allows for dynamic creation of key hierarchies based on the algorithm described in BIP32.
