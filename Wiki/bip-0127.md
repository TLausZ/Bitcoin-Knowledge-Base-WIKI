# BIP-0127: Simple Proof-of-Reserves Transactions

## Summary

This BIP describes a simple way to construct proof-of-reserves transactions. It formalizes a standard format using regular Bitcoin transaction serialization and BIP174 PSBT format, enabling general proof-verification software and easy wallet integration.
