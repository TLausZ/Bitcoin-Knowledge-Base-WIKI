# BIP-0149: Segregated Witness (second deployment)

## Summary

This document specifies a user activated soft fork for BIP-141, BIP-143, and BIP-147 using versionbits with guaranteed lock-in.

## Body

**Layer:** Consensus (soft fork)

### Zweck

BIP-0149 was a fallback activation plan for SegWit, authored by the same person as BIP-148. If BIP-148's August 1, 2017 flag day approached without miner lock-in, BIP-149 offered an alternative: a second BIP-9 deployment of SegWit starting in November 2017, but with a mandatory lock-in — once the signalling window closed, the activation would lock regardless of whether 95% of hashrate had signalled.

This "guaranteed lock-in" mechanism was meant to prevent miners from blocking activation indefinitely through threshold non-compliance, while still giving them a signalling period to coordinate. It became moot when BIP-91 achieved the original BIP-9 threshold in July 2017, before both BIP-148's August 1 deadline and BIP-149's November start.
