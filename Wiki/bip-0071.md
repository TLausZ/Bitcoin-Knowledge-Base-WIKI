# BIP-0071: Payment Protocol MIME types

## Summary

This BIP defines a MIME (RFC 2046) Media Type for Bitcoin payment request messages.

## Body

**Typ:** Specification  
**Layer:** Applications  
**Status:** Deployed (aktiviert und im Einsatz)  
**Themen:** bips, lightning
**Autor(en):** Gavin Andresen  
**Veröffentlicht:** 2013-07-29  
**Quelle:** https://github.com/bitcoin/bips/blob/master/bip-0071.mediawiki

### Zweck

BIP-0071 is a supporting BIP for BIP-70. It registers the MIME types used by the Payment Protocol so that HTTP servers and clients can correctly identify the content type of PaymentRequest, Payment, and PaymentACK messages.

The three types: `application/bitcoin-paymentrequest` (the merchant's signed request), `application/bitcoin-payment` (the customer's signed payment), and `application/bitcoin-paymentack` (the merchant's acknowledgement). Without standardised MIME types, different implementations could serve or parse these messages under different content-type headers, causing interoperability failures.

Largely deprecated alongside BIP-70 itself.
