# BIP-0171: Currency/exchange rate information API

## Summary

A common interface for requesting currency exchange rate information from a server.

## Body

**Typ:** Specification  
**Layer:** Applications  
**Status:** Closed (geschlossen (zurückgezogen oder ersetzt))  
**Autor(en):** Luke Dashjr  
**Veröffentlicht:** 2017-03-04  
**Quelle:** https://github.com/bitcoin/bips/blob/master/bip-0171.mediawiki

### Zweck

BIP-0171 proposed a standardised HTTP API for wallets to query current Bitcoin exchange rates. Instead of each wallet hardcoding a specific exchange's API (with different endpoints, response formats, and authentication), a common interface would let wallets be agnostic about which price feed they use.

The proposal defined request parameters (base currency, target currency) and a JSON response format. It was closed without adoption — the exchange rate data landscape fragmented into numerous competing APIs, and wallets typically either query specific providers directly or use aggregators like CoinGecko. No standard emerged.

## Open Questions
