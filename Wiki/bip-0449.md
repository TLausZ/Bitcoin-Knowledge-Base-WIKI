# BIP-0449: OP_TWEAKADD

## Summary

This BIP proposes OP_TWEAKADD, an opcode that performs x-only elliptic curve point tweaking in tapscript, pushing the tweaked key onto the stack.

### Zweck

OP_TWEAKADD pops a public key (x-only, 32 bytes) and a 32-byte tweak scalar from the stack, computes the tweaked key Q = P + tweak·G, and pushes the resulting x-only key. This exposes Taproot's internal key-tweaking operation (used by BIP-341 to commit to a script tree) as a script-accessible opcode.

With OP_TWEAKADD, a tapscript can verify that a given output's key is the correct tweak of a known base key — enabling covenant patterns that check output structure without OP_CTV's full transaction template. It can also be used to implement pay-to-contract verification (BIP-175 style), verify Taproot commitments within script, and enable more flexible key delegation schemes.

Combined with CHECKSIGFROMSTACK (BIP-348) and OP_INTERNALKEY (BIP-349), OP_TWEAKADD enables scripts that introspect and verify the Taproot construction of outputs.
