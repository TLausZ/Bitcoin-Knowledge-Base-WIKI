# BIP-0047: Reusable Payment Codes for Hierarchical Deterministic Wallets

## Summary

This BIP defines a technique for creating a payment code which can be publicly advertised and associated with a real-life identity without creating the loss of security or privacy inherent to P2PKH address reuse. This BIP is a particular application of BIP43 and is intended to supplement HD wallets which implement BIP44.
