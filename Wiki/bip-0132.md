# BIP-0132: Committee-based BIP Acceptance Process

## Summary

The current process for accepting a BIP is not clearly defined. While BIP-0001 defines the process for writing and submitting a Bitcoin Improvement Proposal to the community it does not specify the precise method for which BIPs are considered accepted or rejected. This proposal sets up a method for determining BIP acceptance. This BIP has two parts: It sets up a process which a BIP goes through for comments and acceptance.
