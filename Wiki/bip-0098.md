# BIP-0098: Fast Merkle Trees

## Summary

In many applications it is useful to prove membership of a data element in a set without revealing the entire contents of that set. The Merkle hash-tree is a cryptographic tool that achieves this goal. Bitcoin uses a Merkle hash-tree construct for committing the transactions of a block into the block header. This particular design, created by Satoshi, suffers from a serious flaw related to duplicate entries (CVE-2012-2459), and also suffers from less than optimal performance due to unnecessary double-hashing.
