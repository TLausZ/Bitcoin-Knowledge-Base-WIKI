# BIP-0081: Hierarchy for Colored Voting Pool Deterministic Multisig Wallets

## Summary

This BIP defines a logical hierarchy for colored coin voting pool deterministic multisig wallets based on BIP-0032 and BIP-0043.

## Body

**Typ:** Informational  
**Layer:** unspecified  
**Status:** Closed (geschlossen (zurückgezogen oder ersetzt))  
**Themen:** bips, self-custody
**Autor(en):** Justus Ranvier, Jimmy Song  
**Veröffentlicht:** 2014-08-11  
**Quelle:** https://github.com/bitcoin/bips/blob/master/bip-0081.mediawiki

### Zweck

BIP-0081 is the "colored coin" companion to BIP-80. Where BIP-80 defined voting pool HD paths for standard Bitcoin UTXOs, BIP-81 defined paths for colored coin assets — tokens issued on top of Bitcoin using the colored coin protocol popular in 2014. The derivation path extended BIP-44 with additional levels for asset types.

Closed for the same reasons as BIP-80: the voting pool concept didn't gain traction, and the colored coin protocols it was designed for (Open Assets, Counterparty) never achieved mainstream Bitcoin wallet adoption.
