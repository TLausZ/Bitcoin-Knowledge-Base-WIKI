# BIP-0080: Hierarchy for Non-Colored Voting Pool Deterministic Multisig Wallets

## Summary

This BIP defines a logical hierarchy for non-colored voting pool deterministic multisig wallets based on BIP-0032 and BIP-0043.

## Body

**Typ:** Informational  
**Layer:** unspecified  
**Status:** Closed (geschlossen (zurückgezogen oder ersetzt))  
**Themen:** bips, self-custody
**Autor(en):** Justus Ranvier, Jimmy Song  
**Veröffentlicht:** 2014-08-11  
**Quelle:** https://github.com/bitcoin/bips/blob/master/bip-0080.mediawiki

### Zweck

BIP-0080 defined HD derivation paths for "voting pool" multisig wallets — a concept from Conformal Systems (now Company 0) where groups of signers manage funds through a formal voting mechanism. The path structure was designed for a specific application architecture rather than general-purpose multisig.

The proposal was closed without finding broad adoption. The "voting pool" concept as originally envisioned did not become a widely used Bitcoin application pattern. General multisig derivation was later standardised more broadly via BIP-45 and BIP-48, which serve the same underlying need without the voting pool framing.
