# BIP-0371: Taproot Fields for PSBT

## Summary

This document proposes additional PSBT fields for BIP-174 PSBTv0 and BIP-370 PSBTv2 to carry Taproot-specific signing data.

## Body

**Typ:** Specification  
**Layer:** Applications  
**Status:** Deployed (aktiviert und im Einsatz)  
**Autor(en):** Ava Chow  
**Veröffentlicht:** 2021-06-21  
**Quelle:** https://github.com/bitcoin/bips/blob/master/bip-0371.mediawiki

### Zweck

BIP-174's original PSBT fields were designed for legacy and SegWit v0 inputs. Taproot (BIP-341/342) introduces new signing data: the internal key, Tapscript leaf hashes, Merkle inclusion proofs, and script-path spending data. None of these fit the existing PSBT input fields.

BIP-371 adds a set of Taproot-specific PSBT fields:
- Taproot key-path signature (a BIP-340 Schnorr signature for key-path spends)
- Taproot script-path leaf hash (which leaf is being used)
- Taproot Merkle inclusion proof (the path from leaf to root)
- Taproot internal key (the untweaked key P)
- Taproot derivation information for the output key

For outputs, BIP-371 adds the internal key and the complete script tree structure, allowing signers to verify change outputs belong to the correct wallet without downloading the full transaction.
