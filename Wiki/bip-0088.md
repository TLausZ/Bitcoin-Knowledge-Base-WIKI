# BIP-0088: Hierarchical Deterministic Path Templates

## Summary

This document describes a format for representing templates that specify the constraints that can be imposed on BIP-32 derivation paths.

### Zweck

BIP-0088 defines a string format for expressing constraints on BIP-32 derivation paths — essentially a template language for paths. A template like `m/44h/0h/0h/{0-1}/{0-50000}` says: "valid paths start with m/44'/0'/0', followed by 0 or 1, followed by any index from 0 to 50000."

This is useful for wallet software that needs to validate user-entered paths, for policy enforcement (e.g. hardware wallets limiting which paths a coordinator can request signatures from), and for documentation. Without a standard template format, each implementation invents its own way to describe "which paths are valid for this wallet."

The format supports hardened and non-hardened steps, ranges (e.g. `{0-9}`), and wildcard steps. It complements descriptor wallets by providing a way to describe path constraints independently of the full wallet policy.
